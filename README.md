# How to run

## Clone the repository
```bash
git clone https://github.com/PImpliesQ/DjangoBackend.git
```

## Build with Docker
```bash
docker build -t django-backend .
```

## Run the container
```bash
docker run -p 8000:8000 django-backend
```
